Open Data API for Researchers

Access real-time and historical AQI data for academic research, analytics, and environmental studies.

API Overview

This Open Data API provides structured access to hyperlocal air quality data collected from distributed IoT sensors deployed across urban streets.

  • • Real-time AQI values
  • • Historical pollution trends
  • • Sensor-level & street-level granularity
  • • JSON-based REST API

Available Endpoints

GET /api/aqi/latest/
GET /api/aqi/history/?from=YYYY-MM-DD&to=YYYY-MM-DD
GET /api/sensors/
GET /api/streets/<street_id>/

All endpoints return structured JSON responses.

Usage & Access Policy

  • • Free access for academic and non-commercial research
  • • Attribution required in published work
  • • Rate limiting applied (1000 requests/day)
  • • API keys may be required for extended usage
